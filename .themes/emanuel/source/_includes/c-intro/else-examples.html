Examples

Example 1
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    int a = 2;
    int b = 4;

    if (b/a == a) {
        printf("That is correct!");
    } else {
        printf("That is incorrect!");
    }

    return EXIT_SUCCESS;
}
This program will print That is correct! as the condition is true.
