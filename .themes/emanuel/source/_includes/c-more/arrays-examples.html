Examples

  1. What if, as part of a complex program, we want some code that generates and stores all of the square numbers less than 100? First, we need to know how many of those numbers there are. 12, 22, ..., 92 gives us 9 square numbers that are less than 102 (100). We can store these numbers in an array of ints: 
    int squares[NUM_SQUARES_UNDER_100];

int num = 1;
int i = 0;

while (i <= NUM_SQUARES_UNDER_100) {
    squares[i] = num * num;
    num++;
    i++;
}
