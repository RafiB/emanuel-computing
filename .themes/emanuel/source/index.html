Welcome to Emanuel Computing!

This course is aimed at students who have a keen interest in the practical application of programming. It may be a bit slow for those who have programming experience, but will cover some important topics.

By the end of this course, you will be able to

  1. Safely use the linux shell
  2. Write scripts to perform basic tasks and tests
  3. Write a program to solve problems in the language C
  4. Manipulate memory using C

Modules

  1. The linux terminal
  2. vim
  3. Intro to C
  4. Bash & Filters
  5. More C
  6. Bash & scripting
  7. Git
  8. Further C (pitfalls, structs, memory, linked lists)

The Rules

Put in the effort and you'll be amazed by how quickly you'll develop new skills.

This is not high school, this is a once-a-week opportunity to leave school behind and learn university-level content in a university-style arena. As such, the course will not be policed - no one is going to chase you to complete a task, reprimand you if you arrive late, or even don't show up.

However, don't be disruptive when you're here. There is no tolerance for bad or inappropriate behaviour.

Tasks will be set. They are not mandatory, but have a go! Remember the golden rule in computing: if you don't know how to do something, google it. If you try (even if you don't succeed), know you're doing awesomely!

Help another student and you get a small reward. Complete all homework problems for 4 weeks in a row and earn a prize (USB stick, earphones, ..)

Any problems or queries, drop me an email and I'll see what I can do to help.

Above all, remember to have fun!
