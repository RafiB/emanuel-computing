Welcome to Emanuel Computing!

This course is designed for students who want to learn how to code. It may be a bit slow for those who have programming experience, but will cover some important topics.

By the end of this course, you will be able to

  1. Safely use the linux shell
  2. Write scripts to perform basic tasks and tests
  3. Write a program to solve problems in the language C
  4. Manipulate memory using C

Modules

  1. The linux terminal (basic commands and what not to do)
  2. Intro to C (program outline, style guide, printing and compiling C, variables, printf args, if/else, while, for)
  3. Bash & Filters
  4. More C (what the program outline means, user input from stdin)
  5. Bash & scripting
  6. Further C (pitfalls, structs, memory, finishing with linked lists)

The Rules
