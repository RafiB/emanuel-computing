Examples

Example 1
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    char myLetter = 'H';

    if (myLetter == 'H') {
        printf("My name starts with H!");
    }

    return EXIT_SUCCESS;
}
This program will print My name starts with H!. Here, the condition is that the variable myLetter is the same as 'H'. The condition is true, so the code inside the braces will run.
Example 2
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    char myLetter = 'H';
    char yourLetter = 'R';

    if (myLetter == yourLetter) {
        printf("Our names start with the same letter!");
    }

    return EXIT_SUCCESS;
}
This program will not print anything. The condition being checked here is that the variable myLetter has the same value as the variable yourLetter. Given this is not true, the code inside the braces will not run.

Note: The condition here looks a little different to those which we are used to. As well as comparing the value of a variable to the value of a number as we have seen in previous examples, we are also able to compare the value of a variable with the value of another variable. Here we are comparing myLetter and yourLetter. This is also completely valid.
