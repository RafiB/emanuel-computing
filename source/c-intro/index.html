Intro to C

The Anatomy of a C Program

When programming in C, every program you write will have the same outline: 
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    // YOUR CODE GOES HERE

    return EXIT_SUCCESS;
}

Note: You can write comments in your program that are ignored when you run it. For one line comments, this is done with a // as above. For multi-line comments, we format the comment

/* Your multi-line
   comment goes
   here */

Style Guide

Style is paramount in writing a readable, understandable program. Group projects and code collaboration are both big parts of professional programming, so well-formatted code is expected and appreciated.

We will teach you C in a clean, readable style, and your personal flair will show up with practice. As long as your code remains nicely formatted, deviations from the styleguide are acceptable.

Structurally speaking, style when programming in C can be the difference between your code working or not working. We will learn more about this in a later module, though you can take a look at the UNSW's C Style Guide if you are interested.

Your first C program

When learning a new programming language, the first thing we do is write a program to display the message "Hello world" on the screen. This will give us an understanding of the basic syntax and structural layout of a program.
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    printf("Hello world!");

    return EXIT_SUCCESS;
}
Don't worry if it doesn't make sense yet - this is what we are here to learn. All you have to notice is this:
When we print something out to the screen, we use printf, a function which takes a string and prints it out to stdout.

At the moment, our program is in a language that we can easily understand. However, the C file alone is not something that the computer can understand. To convert a file to machine language, the C file must go through a process called compilation.

The tool we use for compilation is called GCC. We like to have lots of control over the compilation process, so we do the compilation ourselves from the terminal. First, cd to the directory where you've saved the file. In this example, let's say that the C file with our C code in it is called helloWorld.c. To compile our code, we run: 
$ gcc -Wall -Werror helloWorld.c -o hello
Let's break that command into its components.

When compiling code there are two issues which we can be faced with: warnings and errors. Warnings tell us that something is not quite right with the code. GCC will compile the code despite warnings. Errors are bigger problems. They occur when GCC can't understand your code.

The two flags being used here are -Wall and -Werror. The -Wall flag enables GCC's common warning flags. The -Werror flag treats all warnings as errors. This means that we have a lot more information about our code when debugging.

After these two flags, we then need to tell the compiler which file we want to compile. In this case it is helloWorld.c.

The -o flag tells GCC to output, and following this we specify the name of the file that we want to output the executable code to. Here we call it hello. While the -Wall and -Werror flags can be split up, the output file name must come directly after -o. For example, something like 
$ gcc -Wall -Werror hello -o helloWorld.c
will overwrite your helloWorld.c file with the output. However, 
$ gcc -o hello -Werror helloWorld.c -Wall
will work as we want.

Variables

When programming, we use variables to store information which we use in our program. They are called variables as their values can be changed. In C (unlike some other languages) the type of a variable cannot be changed. We must choose the correct type of variable to represent the data we want to store in it.

Here are some basic types, commonly used in C programs. There are other types, but we tend to prefer the following to their alternatives.

int Integer
char Character
double Double-precision floating point number (decimal)

Placeholders

We can print out the value of a variable with printf like we used in the "Hello world" example. To do this, we must format a string. This means we must reserve a place in the string for the variables. This reservation is called a placeholder.

Every type of variable has its own placeholder:

Type Placeholder
int %d
char %c
double %lf
Here, a placeholder will be used to hold a place for our variable in the string which we are printing. 
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    int myNumber = 5;
    printf("My favourite number is %d\n", myNumber);

    return EXIT_SUCCESS;
}

Control Structures

C runs code line by line from the top of the file. Think about it as a human following a recipe - steps are followed in order. Control structures allow us to repeat, skip, or execute steps when certain conditions are met.

if

A programmer's wife asks him to go out and get a loaf of bread. "Oh, and if they have eggs," she says, "get a dozen!". He returns with twelve loaves of bread.
if allows us to execute code if certain conditions are met. The if statement comprises of the word if, followed by conditions in parentheses. The condition is checked, and if it is met, the code in the braces will run. If the condition is not met, the code will not run. 
#include <stdio.h>
#include <stdlib.h>

    int main (int argc, char* argv[]) {

        int myNumber = 5;

        if (myNumber == 5) {
            printf("My favourite number is five!");
        }

        return EXIT_SUCCESS;
    }

Examples

Example 1
#include <stdio.h>
#include <stdlib.h>

        int main (int argc, char* argv[]) {

            char myLetter = 'H';

            if (myLetter == 'H') {
                printf("My name starts with H!");
            }

            return EXIT_SUCCESS;
        }
This program will print My name starts with H!. Here, the condition is that the variable myLetter is the same as 'H'. The condition is true, so the code inside the braces will run.
Example 2
#include <stdio.h>
#include <stdlib.h>

        int main (int argc, char* argv[]) {

            char myLetter = 'H';
            char yourLetter = 'R';

            if (myLetter == yourLetter) {
                printf("Our names start with the same letter!");
            }

            return EXIT_SUCCESS;
        }
This program will not print anything. The condition being checked here is that the variable myLetter has the same value as the variable yourLetter. Given this is not true, the code inside the braces will not run.

Note: The condition here looks a little different to those which we are used to. As well as comparing the value of a variable to the value of a number as we have seen in previous examples, we are also able to compare the value of a variable with the value of another variable. Here we are comparing myLetter and yourLetter. This is also completely valid.

else

The complement to if is else. This creates a simple either-or situation: if certain conditions are met, do this, otherwise do that. 
#include <stdio.h>
#include <stdlib.h>

        int main (int argc, char* argv[]) {

            int myNumber = 4;

            if (myNumber == 5) {
                printf("My favourite number is five!");
            } else {
                printf("%d is not my favourite number.", myNumber);
            }

            return EXIT_SUCCESS;
        }

else if

else if is a combination of if and else. Like else, else if allows code to be executed if the initial if statement was not run. Unlike else, else if allows us to specify a different condition. 
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {

    int myNumber = 4;

    if (myNumber == 5) {
        printf("My favourite number is five!");
    } else if (myNumber < 5) {
        printf("%d is lower than my favourite number.", myNumber);
    } else {
        printf("%d is higher than my favourite number.", myNumber);
    }

    return EXIT_SUCCESS;
}
Note: In this example, we could have replaced the else statement with

else if (myNumber > 5)

though this is not strictly necessary.

while

A programmer's husband asked her to go and get a loaf of bread. "Oh, and while you're out there," he said, "get some eggs too!" She never came home.
while allows us to execute code while certain conditions are met. This creates a loop. The condition in the while loop is checked, and the loop runs. Before the loop runs again, the condition is checked again. If the condition is no longer met, then the loop will not run. An example of a while loop is: 
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char* argv[]) {
    int i = 1;

    while (i <= 10) {
        printf("Hello!");
        i++;
    }
    
    return EXIT_SUCCESS;
}
This is more concise than ten lines of print statements. We use loops to help us minimise repeated code - the good programmer is the lazy programmer!

Examples

Example 1
#include <stdio.h>
#include <stdlib.h>

    int main (int argc, char* argv[]) {

        int i = 1;

        while (i <= 10) {
            printf("%d\n", i);
            i++;
        }

        return EXIT_SUCCESS;
    }
This program will print out the numbers from 1 to 10, inclusive, with each number on a new line.
Note: The \n in the string puts a new line in after each number printed. The \ is the escape character which tells the compiler that the character(s) following the \ has a special property. The n is the newline character.
