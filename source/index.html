Welcome to Emanuel Computing!

This course is designed for students who want to learn how to code. It may be a bit slow for those who have programming experience, but will cover some important topics.

By the end of this course, you will be able to

  1. Safely use the linux shell
  2. Write scripts to perform basic tasks and tests
  3. Write a program to solve problems in the language C
  4. Manipulate memory using C

Modules

  1. The linux terminal
  2. vim
  3. Intro to C
  4. Bash & Filters
  5. More C
  6. Bash & scripting
  7. Git
  8. Further C (pitfalls, structs, memory, linked lists)

The Rules

Muck around you're out. That goes for behaviour, bullying, or running rm -rf *

This is not high school, I expect a different level of engagement and maturity. This is a once-a-week opportunity to leave school behind and learn university-level content in a university-style arena.

You can be late, you can skip weeks, but don't be disruptive when you're here

There are tasks to do. They aren't homework, but you'll be a lot better off if you do the questions, and I'll know if you haven't done them. If you aren't here to improve, why are you here? BUT if you try and don't succeed, you're doing awesomely! Drop me an email (I'll reply some time after 1000) and I'll see what I can do to help.

Help another student and get yourself a lolly. Complete all homework problems for 4 weeks and earn a prize (USB stick, earphones, ..)

If you don't know how to do something, google it!
