Foreword

Linux is a computer operating system. Linux is a variant of Unix. Indeed, Linux is named after the original developer of Linux; Linus Torvalds.
Mac OS-X, the operating system that your apple laptops are running, is Unix-based. Thus, whenever "Linux" is mentioned, we are really talking about Unix variants. All instructions covered here will work on Mac laptops.

The linux terminal is a hierarchy, or in computer science terms, a tree. The root of the tree, or the highest point in the hierarchy, is called the root directory and is represented simply by /.

Folders in the root directory take the form /<folder-name>, such as /home

Sub-folders are named in the same fashion. My "home directory", the folder assigned to my user, is /home/rafi

And now, it is time to start learning how to navigate around the hierarchy!

To do this, we use the terminal, or console.

Note: in examples concerning the terminal, $ represents the prompt, which is the part of the terminal that waits for the next instruction.

pwd

pwd stands for print working directory. The working directory is the directory that you are currently in.

This is a very handy tool to have when you're using files that are spread across different folders on your computer.

cd

cd stands for change directory. This command is used, as its name might suggest, to move between directories. For example, if you are in your home directory (/home/username) and you want to change to the root directory (/), you would enter the command cd / into the terminal.

ls

ls is the list command. It is used to list the files and sub-directories contained in a directory.

Simply entering ls in the terminal will list the files in the current directory. You can provide options (called arguments) to ls. For example, ls / will list the contents of the root directory.
